# Digit Recognizer 🖊️🔢

A Handwritten Digit Recognition system built using the MNIST dataset.
The application allows users to draw a digit (0–9) on a canvas and predicts the digit using a Convolutional Neural Network (CNN).

---

## 📌 Project Overview

This project demonstrates a complete Machine Learning / Deep Learning pipeline:
- Training a CNN model on MNIST
- Saving the trained model
- Deploying the model using a Flask backend
- Creating an interactive web-based UI for digit drawing and prediction

---

## 🧠 Technologies Used

- Python 3
- TensorFlow / Keras
- Flask
- OpenCV
- NumPy
- HTML, CSS, JavaScript
